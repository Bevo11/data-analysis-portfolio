# Power BI: Telco Customer Churn Analysis

## Project Overview
This project serves as a comprehensive churn analysis dashboard developed as part of the Data Analytics Internship at SaiKet Systems. The goal was to visualize key performance indicators (KPIs) and driver variables that contribute to customer churn in the telecommunications sector.

## Business Problem
The project addresses the challenge of customer retention. By analyzing historical data, we aim to uncover why customers leave and identify high-risk segments based on demographics, tenure, and service usage.

## Key Dashboard Components
- **Churn Rate Overview**: High-level KPI card displaying the total churn percentage.
- **Customer Demographics**: Visual analysis of churn distribution across gender, partner status, and dependents.
- **Tenure Analysis**: Histogram visualization of customer lifetime duration.
- **Churn Drivers**: Heatmaps and stacked bar charts identifying correlations between churn and:
    - Contract Type (Month-to-month vs. Annual)
    - Payment Method
    - Internet Service Provider type

## Technologies Used
- **Power BI**: Primary tool for data modeling and interactive visualization.
- **DAX (Data Analysis Expressions)**: Used for calculating churn rates and custom metrics.
- **Power Query**: Used for data cleaning and transformation to ensure model accuracy.

## Project Files
- `Customer_Churn_Analysis.pbix`: The source Power BI project file.
- `Telco_Customer_Churn_Dataset (2).csv`: The source dataset utilized for modeling.
